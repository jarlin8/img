2. Installation

First, you need to download the plugin package. Login into your CodeCanyon account, go to Downloads section and click Download button next to the DW Question & Answer Pro WordPress plugin.

In the next step, please extract the zip file, then you’ll see these ones (zip file name can be changed from version to version).

Finally, follow these steps below to install the plugin from WordPress Dashboard or via FTP:

2.1. Install via WordPress Dashboard

Step 1: Log into your WordPress Dashboard > Plugins > Add New

Step 2: Click the Upload Plugin button

Step 3: Browse to the dw-question-answer-pro-1.0.0.zip file then click the Install Now button.

Step 4: Once done, click the Activate Plugin link.

2.2. Plugin Installation via FTP

If you are going to install the theme using FTP, you will need an FTP Client, such as FileZilla.

Step 1: Log into your hosting space (server) via FTP client.

Step 2: In extracted archive folder, find dw-question-answer-pro-1.0.0.zip file

Step 3: Extract dw-question-answer-pro-1.0.0.zip file to dw-question-answer-pro folder

Step 4: Upload the dw-question-answer-pro folder to your server in path .../wp-content/plugins/

 

The uploaded path should be like this: .../wp-content/plugins/dw-question-answer-pro/

Step 5: Login to your WordPress Dashboard and go to Appearance > Plugins > Installed Plugins and activate the DW Question & Answer Pro plugin.

Congratulations! You are ready to use DW Question & Answer Pro, check the Getting Started part of our documentation to see what you may want to do after finishing the installation.

3. Getting Started

3.1. Insert Demo Content

Step 1: Log into your WordPress Dashboard.

Step 2: Go to Tools > Import. Then click on WordPress (bottom of the list).

Step 3: In a few seconds, a window will appear. Click the button Install Now, which is at the bottom right of the window.

Step 4: Click Activate plugin & Run Importer.

Step 5: Then choose the dwqa-dummy-data.xml file and click Upload file and import

3.2. Setup Question List & Ask Question Page

Upon installation, DW Question & Answer Pro creates the following new pages:

  • DWQA Questions: The [dwqa-list-questions] shortcode must be on this page.

  • DWQA Ask Question: The [dwqa-ask-question] shortcode must be on this page.

Go to the Questions > Settings > General to select this page for DW Question & Answer Pro.

 

Note: If you choose to customize the template page for DWQA Questions page then the single question pages, the question categories, and the question tags will also use that template to display. If not, these pages will be shown based on the layout of the default pages.

3.4. Setting For Questions

Question Setting section lets you configure settings such as number of posts to show, enabling or disabling private questions.

  • Archive Page Show At Most: Show the largest number of the questions in the questions listing page.

  • Before A Question Appears: Allow you to approve the questions.

  • Other Question Settings: Allow you to post private questions, to enable/disable question status features and status icons on the left side.

3.5. Setting For Answers

Answer Settings section will give you the option to show all answers on a single question page only. Also, it allows you to enter the number of answers you wish to show on the single question page.

3.6. Captcha Settings

3.7. Permalink Settings

You may enter the custom structure for your single question, question category and question tag URLs here. For example, using “topic” as your question base would make your question links like http://your-website-link/topic/question-name/. If you leave these blank, the default will be used.

If your updating makes your site inaccessible. Your posts  and questions turn to a 404 Error page, don’t panic, all your posts are still there and safe. What you need to do is to fix the permalink settings of the DW Question & Answer plugin.

To do this, go to Settings > Permalinks, scroll down and simply click on the Save Changes button.

3.8. Email Notifications Settings

This section allows you to set up notification email sent out to Admin and Users regarding new questions, answers and comments.

Email Settings: you can upload your logo and define which email address will be used as the sender.

New Question Notifications: send out to Administrator regarding new questions posted.

New Answer Notifications: send out to Question’s author or followers regarding new answer posted.

New Comment Notifications: send out to Question’s author regarding new comment posted.

3.9. Permissions Settings

Permission Settings section allows you to control who and how they can interact with your Question & Answer page.

4. Usage Guidelines

4.1. Ask Question

To ask questions, please make sure that you've created the page "Ask question" in the previous step. And the account you are using has been granted permission to ask questions.

At the bottom right of the Questions List page, there will be a button "Ask question" as shown below:

Click this button. You will redirect to the ‘Ask Question page’. Here you can enter your question.

4.2. Manage Question

If you are granted the right to edit / delete questions, or your account is the asking one, you'll see these tools after visiting an article.

When clicking Edit, you'll see a form shown as when asking questions, that you can edit the content of the question.

Also, below the question, at the bottom have a question that you can select to change the status of the question.

4.3. Subscribe Question

This feature helps your users to track all new activities of the question. Whenever there is a new answer or a new comment, they will receive a notification email. At the same time, this question also appears in the tab ‘Subscribe’ of the user’s profile page. The asker is automatically considered as a subscriber of that question. And, if you’re no longer wish to receive the notifications, simply uncheck the Subscribe button.

4.4. Add Answer

Pull down the question, if the user’s account has the right to ask question, there will appear a form answer as below.

Respondents can also set their answers as Private and it is only shown to the admin. This setting is useful in case of the respondents want to send the admin information like his/her admin account or FTP Account.

4.5. Pick The Best Answer

Asker can choose the answer that they think it's the best. This answer will be pin on top of answers, just below the questions to help others can quickly find the best answer for the question. To pick the best one, click the checkbox to the left of the answer.

5. For Developers

5.1. Theme Integration

DW Question Answer Pro plugin provides the markup and template structure for the front-end appearance in the /wp-content/plugins/dw-question-answer-pro/templates directory. However, you should not edit directly in these files because they will be lost whenever you update the plugin.

To override any template file of the DW Question & Answer Pro plugin, please follow our instruction below:

Step 1: Go to the /wp-content/themes/your-theme, create a new folder named “dwqa-templates”

Step 2: Navigate to the /wp-content/plugins/dw-question-answer-pro/templates, select the file you want to override, then copy the file into /wp-content/themes/your-theme/dwqa-templates.

For example: To override the markup of the questions listing page, copy archive-question.php to the /wp-content/themes/your-theme/dwqa-templates.

Now you can edit the /wp-content/themes/your-theme/dwqa-templates/archive-question.php file as you need.

5.2. Translate Plugin

If you are going to translate this plugin, you will need to download Poedit (available for Windows, Mac, and Linux).

Step 1: Open Poedit and select File > New Catalog from POT / PO file or click “Create new translation button” on the first screen

Step 2: Select the dwqa.pot file (you can find this file in the wp-content/plugins/dw-question-answer-pro/languages/ folder).

Step 3: A box will pop up asking for the  language of the translation. Select your language:

Step 4: Hit OK and Poedit will name automatically your file. If you select Spanish (Spain): es_ES.po

 

Step 5: Save your file on your computer

Step 6: Now you can start translating your theme or your plugin. Space at the top will display all the text ready to translate, and any completed translations will display to the right. The boxes underneath show the source text, your translation and any notes for translators. Translating is really easy – just select a word or phrase, enter your translation and click “Validate”.

 

IMPORTANT:

You have to respect all special characters from the original text when translating

It’s not necessary to translate all text strings. Perhaps you don’t want to spend your time translating text strings of the plugin settings.

Step 7: When you’ve finished, save your file. Poedit will automatically create both .po and .mo files. You’ll have to upload both to wp-content/plugins/dw-question-answer-pro/languages folder.

5.3. Available Shortcodes

Used on the questions list page, this shortcode displays all questions and interface for filter, search, pagination, sort, ask question button

[dwqa-list-questions]

Args: none

Ask Question Form

Used on the ask question page, this shortcode displays the ask question form.

[dwqa-ask-question]

Args: none

  • DWQA Ask Form: Show question form.

  • DWQA Featured Questions: Show a list of featured questions.

  • DWQA Closed Questions: Show a list of closed questions.
